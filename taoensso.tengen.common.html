binding-rvals

macro

(binding-rvals bindings)
Evaluates and returns vector of rhs values for given bindings while
preserving support for the usual let-binding facilities like destructuring,
referring to previous bindings, etc.

  [{:keys [x]} {:x 1}
   <...>
  ] ->
  (let [alias1      {:x 1}
        {:keys [x]} alias1
        <...>
       ]
    [alias1 <...>])
